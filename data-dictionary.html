Data Dictionary

This data dictionary provides a comprehensive overview of the data fields used in the RealPage Rent Impact Dashboards, visualizations, and analyses. It serves as a reference for understanding the structure and content of the data, which is essential for interpreting the results and insights presented in the dashboard. It includes descriptions of each field, its data type, and its significance in the context of the dashboard's functionality.

Field Descriptions
Field Name Type Description
propertyId Integer Unique identifier for each property.
propertyName Text Name of the property.
zipCode Text Postal code where the property is located.
address Text Street address of the property.
city Text City in which the property is situated.
state Text Two-letter state code representing the location.
averageSquareFootage Number Average square footage per unit at the property.
stories Number Number of stories in the property building.
stable Text Indicates whether the property’s operational status is stable.
class Text Classification rating of the property, typically represented by a letter (for example, “A,” “B,” or “C”).
latitude Number Geographic latitude of the property.
longitude Number Geographic longitude of the property.
imageHero Text File path or URL for the primary image associated with the property.
heroSource Text Source of the hero image; for example, the name of the image provider.
submarketName Text Submarket or subregion where the property is located.
unitcount Number Total number of residential or commercial units in the property.
yearBuilt Number The year the property was constructed.
siteStatus Text Current status of the property site (for example, “Stabilized”).
type Text Property type or building style (for example, “Conventional”).
MPF-ANN-RENT-CHG Number Annual change in rent, represented as a decimal value.
MPF-HIST-ASK-RENT Number Historical asking rent value; typically, the monthly rent amount in dollars.
MPF-HIST-AVG-ASK-RPSF Number Historical average asking rent per square foot.
MPF-HIST-CONC-RATIO Number or Text Historical concession ratio, indicating any adjustments made to rent (this value may be missing).
MPF-OCC Number Occupancy metric; likely represents the occupancy rate of the property.
MPF-RENT Number Current rent amount at the property.
MPF-RPSF Number Current rent per square foot.
data_index Number Index number assigned to the data row.
cbsa_code Number Core-based statistical area code representing the broader metropolitan area.
